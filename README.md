# quanta-ui

> A Desktop UI Library For Vue.js

## Install

``` bash
# install dependencies
npm install
```

## Import

you can use the full bundle of QuantaUI.

``` javascript
import Vue from 'vue'
import QuantaUI from '../src'

Vue.use(QuantaUI)
```
